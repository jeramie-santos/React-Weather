# React-Weather-App

This project is created to diplay weather information from a specific city/country. It is developed using React and Tailwind.

## Features

- Search - Allow to search a specific city or country.
- Display - Display current weather information of the specified city/country.

## Data
- This app uses openweathermap.org to fetch weather data.

Live Preview: https://react-weather-nine-gray.vercel.app/
